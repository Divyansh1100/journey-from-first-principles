# 🚀 Journey From First Principles

> *A four-year journey through Computer Science, Software Engineering, Artificial Intelligence, and Systems Design — built from first principles.*

---

## 📖 About This Repository

Welcome to **Journey From First Principles**.

This repository is not a collection of class notes.

It is a long-term engineering handbook that documents my journey from a first-year engineering student to a software engineer capable of designing, building, deploying, and maintaining real-world software systems.

Every topic in this repository is studied from **first principles**.

Instead of memorizing technologies, I aim to understand:

* Why a technology exists.
* What problem it solves.
* How it works internally.
* How it is used in real-world systems.
* How to build a simplified version from scratch.

Over the coming years, this repository will continuously grow through projects, experiments, documentation, reflections, and practical implementations.

# 🧠 Learning Philosophy

This notebook follows a simple principle:

> **Understand before memorizing. Build before mastering.**

Every chapter follows the same learning process:

1. Start with a real-world problem.
2. Develop a personal hypothesis.
3. Analyze the strengths and weaknesses of that hypothesis.
4. Understand the engineering solution.
5. Build a simplified implementation.
6. Connect it with real-world technologies.
7. Reflect on the learning.

The objective is not simply to learn technologies, but to develop the ability to learn any technology independently.
